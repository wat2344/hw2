Отчет по ДЗ2: Связный список

Цель: Реализовать программу для работы с 4 типами связных списков на C++.

Выполнено:

Реализованы 4 типа списков: однонаправленный, двунаправленный, кольцевые однонаправленный и двунаправленный
Поддержка аргументов командной строки: -t (тип 1-4), -c (количество), -e (элементы через запятую)
Интерактивное меню с операциями: добавление, вывод, поиск, удаление (первое/все вхождения), очистка, запись/чтение файлов
Работа с файлами с сохранением типа списка
Архитектура:

Абстрактный класс LinkedList
4 класса-наследника для каждого типа списка
Полиморфная обработка операций
Проверки:

Корректность аргументов командной строки
Соответствие количества элементов
Проверка типа при чтении файлов
Управление памятью
Итог: Программа соответствует ТЗ, поддерживает все требуемые функции и типы списков.
